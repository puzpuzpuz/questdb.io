This page describes how to download and run QuestDB via binaries.

- Linux or FreeBSD versions include `questdb.sh` script,
- Windows version has `questdb.exe` executable,
- For macOS, check out [Homebrew](/docs/get-started/homebrew/).

## Download the binaries

Use the following links to download a tarball for your operating system:

## Run QuestDB

This section lists commands to run QuestDB for different OSes. They can be used
to run No JVM binaries.

<!-- prettier-ignore-start -->

<TabsPlatforms
  platforms={[
    { label: "Linux", value: "linux" },
    { label: "Windows", value: "windows" },
    { label: "FreeBSD", value: "bsd" },
  ]}
  render={({ platform }) => {
    if (platform === "linux" || platform === "bsd") {
      return (
        <CodeBlock className="language-shell">./questdb.sh start</CodeBlock>
      );
    }
    if (platform === "windows") {
      return (<RunWindows/>); 
    }
  }}
/>

<!-- prettier-ignore-end -->

The file structure is as the following:

```bash

/questdb
├── conf
├── db
├── log
└── public
```

For information about the file structure, see
[root directory](/docs/concept/root-directory-structure/).

## Upgrade QuestDB version

This section lists commands to upgrade QuestDB for different OSes. They can be
used to run No JVM binaries.

:::note

Check the [release notes](https://github.com/questdb/questdb/releases) and
ensure that necessary [backup](/docs/operations/backup/) is completed.

:::

Steps to upgrade the QuestDB version:

- Stop the instance
- Overwrite the `bin` and `lib` folders with the new files
- Restart the instance

<!-- prettier-ignore-start -->

<TabsPlatforms
  platforms={[
    { label: "Linux", value: "linux" },
    { label: "Windows", value: "windows" },
    { label: "FreeBSD", value: "bsd" },
  ]}
  render={({ platform }) => {
    if (platform === "linux" || platform === "bsd") {
      return (<UpgradeLinux/>);
    }
    if (platform === "windows") {
      return (<UpgradeWindows/>);
    }
  }}
/>

<!-- prettier-ignore-end -->
